Jest runner for cypress.io

## Usage

```sh
npm install --save-dev cypress-jest
```

Your `jest.config.js` should look like this:

```js
module.exports = {
  projects: [
    {
      displayName: 'cypress'
      runner: 'cypress-jest',
      testMatch: ['<rootDir>/**/*-cy.js'],
    },
    {
      displayName: 'test' // Your other unit Tests with jest
    }
  ]
}
```
